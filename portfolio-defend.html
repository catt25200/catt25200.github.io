Project : Defend

Project : Defend is a tower defense game I made after the second semester ended during the summer
I have wanted to make a tower defense game for a very long time.
It is a very fun experience at that time. Using 2D array to build a reactble map.
While making vision limit for this game, I was also thinking a way to let player see more in the dungeon.

*Note this game is still in early development stage
