Project : Defend

Project : Defend is a tower defense game I made after the second semester ended during the summer
Tower defense game like Bloons Tower Defense has brought me a lot of joy. This made me want to make a tower defense game for a very long time.
But the thing is sometime I feel like only placing tower is kinda boring.
Hence I want to improve during the tower defense game I made. By making the player also being a part of the defense.

*Note this game is still in early development stage
