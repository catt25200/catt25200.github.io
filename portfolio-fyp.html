Shrine Keeper

Shrine Keeper is a top-down RPG game which a group project for my computer game development study.
It is set to a father who dies while fighting evil. And this causes those evil forces to try revenge by hurting his son.
Now player as a spirit form of the father needs to protect his son from monsters.
While doing the group project, I feel fun and challenging at the same time.
Not only because it is my first time making a non-mini game. It is also quite a challenge to learn to cooperate with other group members.

*Note this game is still in early development stage

Interactble Object
In this game palyer can interact with different object to gain some advantages.

In this case player can move to a easier path by breaking this tree.

There are some light as a hint for player.

Talent System

There are total 9 talents and and 3 tires for now
Each time can choose only one of the three talents on the same tire.

Player can gain some special ability to prepare more difficult stage later on.

Energy System

Whever player hit an enemy it will increase his energy as the image show.

Shooting an arrow require energy full.

It will show on the top left corner.
