Tea Cup Hour

In recent years, Hong Kong had started to face problems caused by the aging population.
And Tea Cup Hour is a mobile app that aims to help improve the lives of the elderly.
Before making the game, we need to first define what problem does elderly nowadays is facing.
After some research, I find out there are three main problems the elderly are facing.
Which are decreasing physical ability, lack of medical knowledge, and lack of social communication.

Hence the aim of this application is below :
1. Provide elderly correct medical knowledge on common illness
2. Encourage the elderly to participate in memorizing games
3. Create a way to let the elderly connect with their family and friends

*Note this application is still in the early development stage

Game mode
There are currently three gamemodes for now

  • Versus Bot
  • Training
  • Marathon
Versus Bot is a mode in which users will be fighting with a bot and completing higher scores within a limited time.

Training is a mode that has unlimited time to let users train the game they want.

Marathon is a mode in which users will be complete higher scores with a bot in all three game types.

Future Plan:
Add online mode for both veruse and marathon.
Let elderly complete with their family and friends. Create a way to let the elderly connect with them, especially with their grandchildren.

Game Type

There are three game type for current version

  • Mini Quiz
  • Reaction Game
  • Hawkeye

Mini Quiz contains question about symptom of common chronic diseases in elderly.

This can solve the problem of elderly delayed treatment on their illness.

Also can improve elderly's memory by encourage them to remember knowledge and facts from the game.

In Reaction Game there have a button will teleport around.

User need to try their best to click the button.

This can help elderly slowly improve their reaction time and physical sensitivities.

Hawkeye is a game which contain color blind test.

User need to try their best to answer what number they see in the picture.

This can help the elderly to find out they might suffer from Glaucoma or age-related Macular Degeneration now. If they could see the number clearly in the past.

Elderly can share their score to their friend and family by clicking the share button.

There will be default message automatic input for the elderly and they just need to click send.

Or they can input their own message.
