Project : Journey

Journey is a simple text-based RPG game and also one of my homework during the first year study.
I tried to add some special mechanics to make the game more interesting.
Like special items can be bought in shop or even a team since the story is there are three adventurers.
Although team function cannot be implement due to time.
I still made a function to make it feel like those three adventurers are working together.

This game is set in middle age. An evil lord is ruling the north for a very long time.
Luckily there are three brave adventurers try to set the north free.
Player will play as one of the adventurer to go though the dunegon and fight the evil lord.
